CSS Grid

Según la paguina de ( M mdn web doc ): Se entiende que es un método de diseño para compaginar cuadricula con CSS en páginas web con dos dimenciones que permite distribuir su contenido en filas y columnas y que nos facilitan crear diseños complejos.
